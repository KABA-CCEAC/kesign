Layout Grid

While your still open to use the 5px grid defined in the WIMP Guidelines , we recommend giving up a strict 5px pixel finegrained grid.

Instead we recommend using the grid system defined in twitter bootstrap:

  • 940px base - 12 columns
  • responsive - 12 columns
